# Retail Sales Analysis

## Project Overview

This project analyzes retail sales data to uncover insights related to sales trends, profitability, customer behavior, and regional performance.

## Objectives

* Analyze sales and profit trends
* Identify high-value customers
* Detect loss-making products
* Evaluate regional performance
* Understand the impact of discounts

## Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Jupyter Notebook

## Project Structure

* `notebooks/` → Analysis notebook
* `Data/` → Dataset

## Status

Project in progress — exploratory data analysis completed.
